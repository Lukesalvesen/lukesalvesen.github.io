CSS Cheatsheet

Box diagram

Syntax Rule Example
margin: margin collapse occurs when 2 margins overlap. We can set these margins. .content{margin:30px;}
overflow: Deals with content that overlaps. This tag allows the user to see all content clearly and instead of overlapping the content will allow users to scroll or hide content .content{overflow:scroll}
opacity: Allows an image/background-color to be see-through .contents{opacity:0.5;}
background-color: Sets the background color of the selected elements box .contents{background-color: blue;}
background-image: This tag allows you to set a background for the chosen element .contents{background-image: url(".....");}
z-inex: z-index brings an element forward or back in the web page making the element move infront or behind something. .contents{zindex:10;}
display: This refers to the elements characteristics. We can choose between block, inline and inline-block. .contents{display:inline-block;}
position: This tag allows us to place a HTML element on the web page. We can use static, relative, absolute or fixed. Fixed will pin the element to the position on the web page like a naviagtion tab. .contents{position:fixed;}
float: This tag allows us to float an element left or right of its original position. a width must be set otherwise it will use its entire width. .contents{float:right; width:40px;}
clear: This shows how an element should reat when bumping into another element. We can clear left or right. .contents{clear:left;}
